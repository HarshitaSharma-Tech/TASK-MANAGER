TASK MANAGER

TASK SYNOPSIS

How to Use the Todo List:

  1. Type the task in the input box.
  2. Click the "Add Task" button or press Enter to add the task to the list.
  3. Click on a task to mark it as done or undo the mark.
  4. Click the "Clear Completed" button to remove all completed tasks.
  5. Click the "Clear All" button to remove all tasks from the list.
  6. Click the "Sort Tasks" button to move completed tasks to the bottom of the list.
